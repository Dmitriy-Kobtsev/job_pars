# Job Sites Parser
Программа поиска и фильтрации вакансий hh.ru и superjob.ru.
## System requirements:
python 3.9
## Setup instruction:
```bash
$ pip install -r requirements.txt
```
## How to run:
$ python main.py
Реализован сценарий взаимодействия с пользователем.
По введенному поисковому запросу, отбираются вакансии с сайтов hh.ru и superjob.ru.
Отобранные вакансии сохраняются в соответствующие json-файлы.
После возможно провести фильтрацию вакансий по введенным ключевым словам и уровню заработной платы.
Интересующие вакансии выводятся отсортированным по уменьшению ЗП списком в консоль.
